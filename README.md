# 🛍 Stylish – iOS E-commerce App

Stylish is a modern e-commerce iOS application built using SwiftUI, designed to simulate a real-world shopping experience with clean architecture and scalable code structure.

---

## 🚀 Features

- 🛍 Browse products with dynamic data
- 🔍 Search and filter products
- 📄 View detailed product information
- 🛒 Add to cart and manage cart items
- ❤️ Wishlist / Favorites functionality
- 🔄 Pagination for large product lists
- ⚠️ Proper error handling and loading states
- 🌙 Dark mode support (if implemented)

---

## 🧠 Architecture

The app follows **MVVM (Model-View-ViewModel)** architecture:

- **Model** → Data structures and API models  
- **View** → SwiftUI views (UI layer)  
- **ViewModel** → Business logic and state management  

This ensures:
- Clean separation of concerns  
- Better testability  
- Scalable codebase  

---

## 🛠 Tech Stack

- Swift  
- SwiftUI  
- MVVM Architecture  
- URLSession (Networking Layer)  
- Codable (JSON Parsing)  
- CoreData / UserDefaults (if used for persistence)
